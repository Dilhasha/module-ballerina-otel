MODIFICATION NOTICE (Apache License 2.0, section 4(b))
--------------------------------------------------------------------------------
This jar is NOT the original org.jetbrains.kotlin:kotlin-stdlib:1.9.23 artifact.
It is a size-reduced ("slim") variant produced with ProGuard
(https://www.guardsquare.com/proguard) by the Ballerina OpenTelemetry extension
(org.ballerinalang:otel).

Changes applied to the original artifact:
  - Only the classes reachable from the OkHttp/Okio libraries (the sole reason
    kotlin-stdlib is bundled) are retained; all other classes are removed.
  - The multi-release entries (META-INF/versions/**) and module-info.class are
    removed, and the stale "Multi-Release: true" manifest attribute is dropped.
  - No class was optimized or obfuscated (ProGuard `-dontoptimize
    -dontobfuscate`); every surviving class is byte-identical to the
    corresponding class in the original artifact.

The reproduction of this procedure is described in the `size-reduction-test`
subproject of the source repository.
